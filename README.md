# Advanced CrossFit Checklist Web App with Report

Included:
- index.html — tracker + report + CSV export
- manifest.webmanifest
- service-worker.js
- icon.svg
- README.md

Deploy on GitHub Pages:
1. Create a new GitHub repository.
2. Upload all files to the repo root.
3. Go to Settings > Pages.
4. Choose Deploy from a branch.
5. Select main and /root.
6. Save and wait for the live URL.

Use on iPhone:
1. Open the live URL in Safari.
2. Tap Share.
3. Tap Add to Home Screen.

Generate Report creates a printable summary page you can save as PDF at the end of the cycle.
